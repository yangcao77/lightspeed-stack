## 4. Core Capabilities

- **Multi-provider LLM access** — OpenAI, Azure, VertexAI, WatsonX, Bedrock, vLLM (RHOAI/RHAIIS/RHEL AI) 
- **RAG** — Inline FAISS, OKP/Solr, tool-based file search
- **Tool Calling** — MCP server integration
- **A2A Protocol** 
- **Streaming**
- **Guardrails**
- **Authentication & Authorization** — 6 pluggable auth modules (K8s, JWT, API key, RH Identity), RBAC with 30+ actions
- **Quotas & Observability** — Per-user and cluster-wide token limits, scheduled resets, Prometheus metrics for API, quota
- **Conversation Management** — Full conversation history storage and retrieval, multiple cache backends (PostgreSQL, SQLite, in-memory)
